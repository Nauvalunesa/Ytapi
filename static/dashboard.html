Panduan Penggunaan API

1. Menggunakan Bearer Token

curl -H "Authorization: Bearer YOUR_TOKEN" https://api.example.com/search/?query=test

2. Menggunakan API Key

curl -H "X-API-Key: YOUR_API_KEY" https://api.example.com/search/?query=test

3. Menggunakan IP Address (jika terdaftar)

curl https://api.example.com/search/?query=test

Endpoint Utama:

  • /search/ - Pencarian video YouTube
  • /download/ - Download video YouTube
  • /download/audio - Download audio YouTube
  • /spotify/search - Pencarian Spotify
  • /spotify/download/audio - Download audio Spotify
